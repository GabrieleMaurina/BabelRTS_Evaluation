This folder contains the scripts used in the experiments for BabelRTS as well as the implementation of BabelRTS.
Additionally, the data collected and plots are also included in this folder.
The folder plot_utils contains some code that is used in all 3 experiments. plot_utils should not be run independently.
All the code was tested using Python3.9
Install the Python dependencies with: "python3.9 -m pip install --user -r requirements.txt"
Additionally, git is required.
